# Photography Portfolio

A responsive photography portfolio website built using HTML, Tailwind CSS, and JavaScript.

## Live Site
https://samannay07.github.io/Photography-Portfolio/

## Features
- Clean and minimal UI
- Dark/Light Toggle Button
- Show/Hide Galley option
- Responsive layout
- Smooth transitions
- Optimized image loading

## Tech Stack
- HTML
- Tailwind CSS
- JavaScript


## Deployment
Hosted on GitHub Pages.
